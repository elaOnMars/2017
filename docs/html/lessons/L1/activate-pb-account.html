Activating Pebbles accountÂ¶

The first time using the course environment, it is necessary to activate your Pebbles account and create your password. This needs to be done only once when the course starts.

Note

If you already have activated your account you can continue with launching a new computer instance.

To activate your account, follow these steps:

1. Follow the activation link that was sent to you by the course lecturer

2. Handle insecure connection warning

This exception will occur only once when you activate your account, hence, you won’t see this message again later.

Handle insecure connection warning by adding an exception

3. Create a password for your account (Be sure to remember this, you will need it frequently)

Create password

4. Log into the Pebbles dashboard

Log into PB

5. Open the “Account” tab

When you first enter the Pebbles system, you have a following warning. Let’s deal with it by going to the “Account” tab.

6. Join to AutoGIS group

Everyone will join to dedicated group for this course by adding a code autogis17-a47rl into the pop-up window.

7. Create SSH keys

Scroll down the page and you will see following section where you should Generate Keypair:

You don’t need to download the generated Keypair into your computer.

8. Return to “Dashboard” tab

Now you are ready to go with launching your first computer instance.
